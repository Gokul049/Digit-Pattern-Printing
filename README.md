# 🔢 Java Digit Pattern Programs (0–9)

This repository contains Java programs that print **digit patterns (0–9)** using the `*` (asterisk) character. Each digit is implemented as a separate Java program to demonstrate the use of **nested loops**, **conditional statements**, and **pattern programming**.

## 📌 Features

- ✅ Digit patterns from **0 to 9**
- ✅ Implemented using Java
- ✅ Uses nested `for` loops
- ✅ Beginner-friendly pattern programming examples
- ✅ Easy to understand and modify
